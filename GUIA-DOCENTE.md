## 🚀 Cómo Usar los Recursos en Clase

### Opción 1: Proyector + Navegador (Recomendado)
1. Abrir `clases-html/index.html` en el navegador
2. Navegar a la clase correspondiente
3. Mostrar la teoría y ejemplos de código
4. Cambiar a VSCode para demostraciones prácticas

### Opción 2: VSCode + Live Server
1. Instalar extensión "Live Server"
2. Clic derecho en `index.html` → "Open with Live Server"
3. Navegar por las clases
