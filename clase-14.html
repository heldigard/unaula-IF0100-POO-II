1. Fundamentos de Archivos Planos

Archivos planos = Archivos de texto que almacenan datos sin un motor de base de datos. Son legibles por humanos y editables con cualquier editor de texto.

¿Cuándo usar archivos planos?

Casos de uso adecuados:

  • Configuraciones de aplicación
  • Logs y auditorías
  • Exportación de datos (CSV, JSON)
  • Caché simple
  • Aplicaciones sin acceso a DBMS

Evitar para:

  • Grandes volúmenes de datos
  • Múltiples usuarios concurrentes
  • Necesidad de consultas complejas
  • Alta frecuencia de escritura

Comparación: Archivo Plano vs Base de Datos

Caso Archivo Plano Base de Datos
Pocos datos ✅ Recomendado ❌ Sobredimensionado
Multiusuario ❌ No adecuado ✅ Recomendado
Consultas complejas ❌ Limitado ✅ Recomendado
Portabilidad ✅ Recomendado ⚠️ Complejo

2. System.IO - API Base de Archivos

System.IO = Namespace de .NET que contiene clases para leer/escribir archivos y directorios (File, Directory, Stream).

LINQ = Sintaxis de C# para consultar colecciones (como SQL pero en código).

Generics = Clases que pueden trabajar con cualquier tipo (List<T>, Dictionary<K,V>).

Serialización = Convertir un objeto en memoria a un formato que pueda guardarse en archivo o transmitirse por red.

Clases Principales

Clase Propósito Uso Típico
File Métodos estáticos para archivos Operaciones simples (leer/escribir todo)
Directory Métodos estáticos para directorios Crear, listar, eliminar carpetas
StreamReader Lectura de texto Lectura línea por línea
StreamWriter Escritura de texto Escritura con buffering
FileStream Lectura/escritura binaria Archivos binarios, locking, acceso aleatorio
Errores Comunes con Archivos
  • Olvidar using: El archivo queda abierto (file lock) y otros procesos no pueden accederlo
  • Encoding incorrecto: Caracteres especiales (ñ, tildes) se corrompen
  • Path sin validar: File.Exists antes de abrir para evitar excepciones
  • Permisos insuficientes: Excepción "Access Denied" si no hay permisos de escritura

Conceptos Clave

FileStream = Permite acceder a cualquier posición del archivo (seek), ideal para archivos binarios o acceso aleatorio.

Encoding = Mapa de caracteres a bytes. UTF-8 es el estándar web y soporta todos los caracteres del mundo.

Buffering = StreamWriter/Reader usan buffer interno para reducir llamadas al disco (escribe/lee bloques en lugar de byte por byte).

Operaciones Básicas

using System.IO;

// Escribir texto completo
File.WriteAllText("datos.txt", "Hola UNAULA");

// Leer texto completo
string contenido = File.ReadAllText("datos.txt");

// Agregar texto al final
File.AppendAllText("log.txt", "Nueva entrada\n");

// Verificar si existe
if (File.Exists("datos.txt"))
{
    // El archivo existe
}

// Leer todas las líneas
string[] lineas = File.ReadAllLines("archivo.txt");

StreamReader y StreamWriter

using System.IO;

// Escribir con StreamWriter (mejor para archivos grandes)
using (StreamWriter writer = new StreamWriter("salida.txt"))
{
    writer.WriteLine("Línea 1");
    writer.WriteLine("Línea 2");
    // El using cierra automáticamente el archivo
}

// Leer con StreamReader
using (StreamReader reader = new StreamReader("entrada.txt"))
{
    string linea;
    while ((linea = reader.ReadLine()) != null)
    {
        Console.WriteLine(linea);
    }
}

3. Persistencia con CSV

Formato CSV

CSV = Formato de texto plano para datos tabulares donde cada línea es un registro y los campos se separan por comas (u otro delimitador).

Es el formato más simple para intercambiar datos con Excel y otras herramientas de hojas de cálculo.

CSV: El estándar de facto para importación/exportación de datos tabulares.

Ejemplo de Archivo CSV

Id,Nombre,Precio,Categoria
1,Teclado,120000,Periféricos
2,Mouse,80000,Periféricos
3,Monitor,450000,Pantallas

Repository CSV - Estructura Base

using System;
using System.Collections.Generic;
using System.IO;
using System.Linq;

public class EstudianteRepositoryCSV
{
    private readonly string _filePath;
    private const char SEPARADOR = ',';

    public EstudianteRepositoryCSV(string filePath = "estudiantes.csv")
    {
        _filePath = filePath;

        // Crear archivo con encabezado si no existe
        if (!File.Exists(_filePath))
        {
            File.WriteAllText(_filePath,
                "Id,Codigo,Nombre,Apellido,Email,Promedio,Activo\n");
        }
    }

    public void Crear(Estudiante estudiante)
    {
        // Generar ID automático
        var estudiantes = ObtenerTodos();
        estudiante.Id = estudiantes.Any()
            ? estudiantes.Max(e => e.Id) + 1
            : 1;

        // Validar código único
        if (estudiantes.Any(e => e.Codigo == estudiante.Codigo))
            throw new Exception($"Ya existe estudiante con código {estudiante.Codigo}");

        // Agregar al final del archivo
        string linea = EstudianteACSV(estudiante);
        File.AppendAllText(_filePath, linea + "\n");
    }

    public List<Estudiante> ObtenerTodos()
    {
        var estudiantes = new List<Estudiante>();
        string[] lineas = File.ReadAllLines(_filePath);

        // Saltar encabezado
        for (int i = 1; i < lineas.Length; i++)
        {
            if (string.IsNullOrWhiteSpace(lineas[i]))
                continue;

            try
            {
                var estudiante = CSVAEstudiante(lineas[i]);
                if (estudiante.Activo)
                    estudiantes.Add(estudiante);
            }
            catch (Exception ex)
            {
                Console.WriteLine($"Error en línea {i}: {ex.Message}");
            }
        }

        return estudiantes;
    }

    public Estudiante ObtenerPorId(int id)
    {
        return ObtenerTodos().FirstOrDefault(e => e.Id == id);
    }

    private string EstudianteACSV(Estudiante e)
    {
        return $"{e.Id}{SEPARADOR}" +
               $"{e.Codigo}{SEPARADOR}" +
               $"\"{e.Nombre}\"{SEPARADOR}" +
               $"\"{e.Apellido}\"{SEPARADOR}" +
               $"{e.Email}{SEPARADOR}" +
               $"{e.Promedio:F2}{SEPARADOR}" +
               $"{(e.Activo ? "1" : "0")}";
    }

    private Estudiante CSVAEstudiante(string linea)
    {
        var campos = SepararCSV(linea);
        if (campos.Length < 7) throw new FormatException("Línea CSV incompleta");

        return new Estudiante
        {
            Id = int.Parse(campos[0]),
            Codigo = campos[1],
            Nombre = campos[2].Trim('"'),
            Apellido = campos[3].Trim('"'),
            Email = campos[4],
            Promedio = decimal.Parse(campos[5]),
            Activo = campos[6] == "1"
        };
    }
}

Manejo de Comillas en CSV

// Separador que respeta comillas
private string[] SepararCSV(string linea)
{
    var campos = new List<string>();
    bool dentroComillas = false;
    string campoActual = "";

    foreach (char c in linea)
    {
        if (c == '"') dentroComillas = !dentroComillas;
        else if (c == SEPARADOR && !dentroComillas)
        {
            campos.Add(campoActual);
            campoActual = "";
        }
        else
            campoActual += c;
    }

    campos.Add(campoActual);
    return campos.ToArray();
}

4. Persistencia con JSON

System.Text.Json

JSON = Formato ligero de intercambio de datos, legible para humanos y nativo de JavaScript. Es el estándar de facto para APIs web.

Ventajas de JSON sobre XML: Más ligero, más fácil de parsear, soporte nativo en JavaScript.

Serialización Básica

using System.Text.Json;

// Serializar objeto a JSON
var producto = new Producto { Id = 1, Nombre = "Teclado", Precio = 120000 };
string json = JsonSerializer.Serialize(producto);
// {"Id":1,"Nombre":"Teclado","Precio":120000}

// Serializar con formato legible
var options = new JsonSerializerOptions { WriteIndented = true };
string jsonBonito = JsonSerializer.Serialize(producto, options);

// Deserializar
var producto2 = JsonSerializer.Deserialize<Producto>(json);

// Serializar lista
var productos = new List<Producto> { producto1, producto2 };
string jsonArray = JsonSerializer.Serialize(productos, options);

Repository JSON - Estructura Base

using System;
using System.Collections.Generic;
using System.IO;
using System.Linq;
using System.Text.Json;

public class EstudianteRepositoryJSON
{
    private readonly string _filePath;
    private readonly JsonSerializerOptions _jsonOptions;

    public EstudianteRepositoryJSON(string filePath = "estudiantes.json")
    {
        _filePath = filePath;
        _jsonOptions = new JsonSerializerOptions
        {
            WriteIndented = true,
            PropertyNamingPolicy = JsonNamingPolicy.CamelCase
        };

        // Crear archivo vacío si no existe
        if (!File.Exists(_filePath))
        {
            File.WriteAllText(_filePath, "[]");
        }
    }

    public void Crear(Estudiante estudiante)
    {
        var estudiantes = ObtenerTodos();

        // Generar ID automático
        estudiante.Id = estudiantes.Any()
            ? estudiantes.Max(e => e.Id) + 1
            : 1;

        estudiantes.Add(estudiante);
        GuardarTodos(estudiantes);
    }

    public List<Estudiante> ObtenerTodos()
    {
        try
        {
            string json = File.ReadAllText(_filePath);
            var estudiantes = JsonSerializer.Deserialize<List<Estudiante>>(json, _jsonOptions);
            return estudiantes?.Where(e => e.Activo).ToList()
                ?? new List<Estudiante>();
        }
        catch (JsonException ex)
        {
            Console.WriteLine($"Error al leer JSON: {ex.Message}");
            return new List<Estudiante>();
        }
    }

    public bool Actualizar(Estudiante estudiante)
    {
        var estudiantes = LeerTodos();
        var index = estudiantes.FindIndex(e => e.Id == estudiante.Id);

        if (index == -1) return false;

        estudiantes[index] = estudiante;
        GuardarTodos(estudiantes);
        return true;
    }

    private List<Estudiante> LeerTodos()
    {
        string json = File.ReadAllText(_filePath);
        return JsonSerializer.Deserialize<List<Estudiante>>(json, _jsonOptions)
            ?? new List<Estudiante>();
    }

    private void GuardarTodos(List<Estudiante> estudiantes)
    {
        string json = JsonSerializer.Serialize(estudiantes, _jsonOptions);
        File.WriteAllText(_filePath, json);
    }
}

5. Persistencia con XML

System.Xml.Serialization

XML = Lenguaje de marcado para estructurar datos con etiquetas jerárquicas. Es más pesado que JSON pero soporta schemas y metadatos.

XmlSerializer = Clase que convierte objetos C# a XML y viceversa, usando atributos para configurar la serialización.

¿Cuándo usar XML en lugar de JSON?
  • Necesitas schemas: XML tiene XSD para validar estructura
  • Sistemas legacy: Muchos sistemas empresariales usan SOAP/XML
  • Metadatos: XML es mejor para documentos complejos con atributos
using System;
using System.Collections.Generic;
using System.IO;
using System.Xml.Serialization;

public class EstudianteRepositoryXML
{
    private readonly string _filePath;
    private readonly XmlSerializer _serializer;

    public EstudianteRepositoryXML(string filePath = "estudiantes.xml")
    {
        _filePath = filePath;
        _serializer = new XmlSerializer(typeof(List<Estudiante>));
        if (!File.Exists(_filePath)) GuardarTodos(new List<Estudiante>());
    }

    public void Crear(Estudiante estudiante)
    {
        var estudiantes = ObtenerTodos();
        estudiante.Id = estudiantes.Any() ? estudiantes.Max(e => e.Id) + 1 : 1;
        estudiantes.Add(estudiante);
        GuardarTodos(estudiantes);
    }

    public List<Estudiante> ObtenerTodos()
    {
        using (var stream = File.OpenRead(_filePath))
        {
            var estudiantes = (List<Estudiante>)_serializer.Deserialize(stream);
            return estudiantes.Where(e => e.Activo).ToList();
        }
    }

    private void GuardarTodos(List<Estudiante> estudiantes)
    {
        using (var stream = File.Create(_filePath))
            _serializer.Serialize(stream, estudiantes);
    }
}

7. Temas Avanzados

Archivos de Configuración INI

INI = Formato simple de archivos de configuración con secciones entre corchetes [Sección] y pares clave=valor.

// Archivo config.ini:
; Archivo de configuración de UniversidadApp

[Database]
Server=localhost
Port=1433
Database=UniversidadDB
UseSSL=true

[App]
Theme=Dark
Language=es-CO
PageSize=20
public class IniFile
{
    private readonly string _filePath;
    private readonly Dictionary<string, Dictionary<string, string>> _data;

    public string GetValue(string section, string key, string defaultValue = "")
    {
        if (_data.ContainsKey(section) && _data[section].ContainsKey(key))
            return _data[section][key];
        return defaultValue;
    }

    public void SetValue(string section, string key, string value)
    {
        if (!_data.ContainsKey(section))
            _data[section] = new Dictionary<string, string>();
        _data[section][key] = value;
    }
}

// Uso
var config = new IniFile("config.ini");
var servidor = config.GetValue("Database", "Server", "localhost");
var puerto = config.GetInt("Database", "Port", 1433);

Compresión de Archivos

GZIP = Formato de compresión que reduce el tamaño de archivos individuales. Es estándar en Linux y para archivos web (.gz).

ZIP = Contenedor que puede almacenar múltiples archivos comprimidos. Es el formato estándar para distribuciones de software.

¿Por qué comprimir archivos?
  • Ahorro espacio: Archivos de texto pueden reducirse hasta 80%
  • Más rápido: Transferir 1MB comprimido es más rápido que 10MB sin comprimir
  • Backups: Comprimir logs antiguos para ahorrar disco

GZIP - Compresión individual

using System.IO.Compression;

public static void CompressGzip(string sourcePath, string compressedPath)
{
    using var originalStream = File.OpenRead(sourcePath);
    using var compressedStream = File.Create(compressedPath);
    using var gzipStream = new GZipStream(compressedStream, CompressionMode.Compress);

    originalStream.CopyTo(gzipStream);

    var ratio = (1 - (double)compressedStream.Length / originalStream.Length) * 100;
    Console.WriteLine($"Ratio de compresión: {ratio:F1}%");
}

// Crear ZIP con múltiples archivos
public static void CreateZip(string zipPath, params string[] filesToInclude)
{
    using var archive = ZipFile.Open(zipPath, ZipArchiveMode.Create);

    foreach (var filePath in filesToInclude)
    {
        if (!File.Exists(filePath)) continue;
        var entryName = Path.GetFileName(filePath);
        archive.CreateEntryFromFile(filePath, entryName);
    }
}

Logger de Auditoría

public class FileLogger
{
    private readonly string _logPath;

    public FileLogger(string logPath = "auditoria.log")
    {
        _logPath = logPath;

    public void Log(string operacion, string detalles, string usuario)
