## Estructura Sugerida de Estudio

### Durante la clase (90 min)
1. Sigue las diapositivas Marp
2. Cuando veas un enlace, abre el material complementario
3. Copia y ejecuta el codigo de `codigo.md` en tu IDE
4. Resuelve los ejercicios de `ejercicios.md`

### Despues de clase (refuerzo)
1. ReLee `teoria.md` para profundizar conceptos
2. Completa los ejercicios adicionales
3. Explora `referencias.md` para ampliar horizontes
4. Trabaja en la tarea para casa

## Temas Cubiertos en esta Clase

### Fundamentos
- Historia de C# - De 2000 a 2026
- Ecosistema .NET - Framework vs Core vs .NET 8
- Arquitectura .NET - CLR, BCL, JIT

### Tipos y Datos
- Value vs Reference Types - Stack vs Heap
- Nullable Types - Cuándo y por qué
- Strings - Inmutabilidad y StringBuilder

### Modern C# (v12)
- Primary Constructors - Sintaxis concisa
- Collection Expressions - Operador spread
- Pattern Matching - Switch expressions

### Practica
- Ejercicios de clase - Calculadora, Calificaciones
- Tarea para casa - Proyecto Calculadora de Promedios
