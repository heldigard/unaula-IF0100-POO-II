## Temas Cubiertos en esta Clase

### Fundamentos POO
- Programacion Orientada a Objetos - Paradigma vs Procedural
- Tres Pilares de POO - Encapsulamiento, Herencia, Polimorfismo

### Clases y Objetos
- Clase vs Objeto - Plantilla vs Instancia
- Representacion en Memoria - Stack vs Heap
- Anatomia de una Clase - Estructura completa

### Encapsulamiento
- Que es Encapsulamiento - Ocultar datos, exponer comportamiento
- Modificadores de Acceso - public, private, protected, internal
- Propiedades en C# - get, set, validacion

### Constructores
- Tipos de Constructores - Default, parametrizado, object initializer
- Encadenamiento con `this` - Constructor chaining

### Avanzado
- Static vs Instance - Miembros compartidos
- class vs struct - Reference vs Value types
