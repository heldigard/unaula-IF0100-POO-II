# Ejercicios - Domain Driven Design

**IF0100 - Lenguaje de Programación OO II**

---

## Ejercicio 1: Banco

Modelar usando DDD:
- **Entidades**: Cuenta, Cliente
- **Value Objects**: Dinero, TipoCuenta
- **Agregado**: Cuenta con sus Transacciones

---

## Ejercicio 2: E-commerce

Identificar:
- Bounded Contexts (Ventas, Inventario, Envíos)
- Entidades vs Value Objects
- Agregados y sus Roots

---

## Ejercicio 3: Sistema Reservas

Implementar:
- Reserva como Agregado Root
- Habitación como Entidad
- FechaRango como Value Object
