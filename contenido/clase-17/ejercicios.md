## Proyecto Final: Sistema Universitario

### Requisitos

#### Funcionalidades

1. **CRUD Estudiantes**
   - Listar todos
   - Crear nuevo
   - Editar existente
   - Eliminar

2. **Inscripción de Materias**
   - Ver materias disponibles
   - Inscribir estudiante
   - Ver inscripciones
   - Registrar notas

3. **Reportes**
   - Estudiantes por carrera
   - Materias con más inscripciones
   - Promedio de notas

#### Técnicos

- **POO**: Clases, herencia, polimorfismo
- **TDD**: Pruebas unitarias xUnit
- **ASP.NET Core**: Razor Pages o MVC
- **ADO.NET**: SQL Server con parámetros
- **HTML5/Bootstrap**: UI responsiva

### Entregables

- [ ] Código fuente completo
- [ ] Script SQL de base de datos
- [ ] Mínimo 10 pruebas unitarias
- [ ] README con instrucciones
- [ ] Capturas de pantalla
