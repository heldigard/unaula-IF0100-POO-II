## Objetivos de Aprendizaje

Al finalizar esta clase, podras:

1. Diferenciar sobrecarga vs sobreescritura de metodos
2. Implementar sobrecarga de metodos y constructores
3. Sobrecargar operadores en C# para tipos personalizados
4. Modelar bases de datos con diagramas Entidad-Relacion
5. Identificar relaciones 1:1, 1:N, N:M y su mapeo a objetos
