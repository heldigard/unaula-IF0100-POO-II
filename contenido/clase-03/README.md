## Objetivos de Aprendizaje

Al finalizar esta clase, podras:

1. Aplicar herencia entre clases usando el operador `:`
2. Usar la palabra clave `base` para llamar constructores y metodos de la clase padre
3. Implementar polimorfismo con `virtual` y `override`
4. Crear clases y metodos abstractos para forzar diseño en clases derivadas
5. Aplicar el principio LSP (Liskov Substitution Principle)
