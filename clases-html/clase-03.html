1. Teor√≠a: ¬øQu√© es la Herencia?

Definici√≥n y Concepto Fundamental

La herencia es uno de los pilares fundamentales de la Programaci√≥n Orientada a Objetos. Permite definir una nueva clase (subclase o clase derivada) que reutiliza, extiende y modifica el comportamiento de una clase existente (superclase o clase base).

La herencia modela una relaci√≥n jer√°rquica donde la subclase "es un tipo de" la superclase. Esta relaci√≥n debe ser natural y l√≥gica:

  • ‚úÖ Un Gerente es un Empleado
  • ‚úÖ Un Perro es un Animal
  • ‚úÖ Un C√≠rculo es una Figura
  • ‚ùå Un Auto NO es un Motor (aqu√≠ va composici√≥n, no herencia)

3. Principio de Sustituci√≥n de Liskov (LSP)

Definici√≥n Formal

El Principio de Sustituci√≥n de Liskov (LSP), formulado por Barbara Liskov en 1987, establece que:

"Si S es un subtipo de T, entonces los objetos de tipo T pueden ser reemplazados por objetos de tipo S sin alterar ninguna de las propiedades deseables del programa."

En T√©rminos Pr√°cticos

Una clase derivada debe poder sustituir a su clase base sin que el programa falle o se comporte de manera inesperada.

@dataclass
class Empleado:

    id: int
    salario_base: float

        print(f"{self.nombre} esta trabajando.")

    def mostrar_info(self) -> None:
        super().mostrar_info()
        print("Cargo: Gerente")
        print(f"Subordinados: {self.subordinados}")
        print(f"Bono: ${self.bono:.2f}")
        print(f"Salario Total: ${self.calcular_salario():.2f}")

        super().mostrar_info()
        print("Cargo: Desarrollador")

        print(f"Horas Extra: {self.horas_extra}")

from abc import ABC, abstractmethod

# Definici√≥n de clases abstractas (interfaces en Python)
class Volador(ABC):
    @abstractmethod
    def despegar(self) -> None: pass

    @abstractmethod
    def aterrizar(self) -> None: pass


class Nadador(ABC):
    @abstractmethod
    def nadar(self) -> None: pass


# Clase que hereda de una clase e implementa m√∫ltiples interfaces
class Animal:
    def hacer_sonido(self) -> None:
        print("Sonido de animal")


class Pato(Animal, Volador, Nadador):
    altura_maxima: float = 1000.0  # metros
    profundidad_maxima: float = 2.0  # metros

    def hacer_sonido(self) -> None:
        print("¬°Cuac cuac!")

    def despegar(self) -> None:
        print("El pato est√° despegando...")

    def aterrizar(self) -> None:
        print("El pato est√° aterrizando en el agua...")

    def nadar(self) -> None:
        print("El pato est√° nadando graciosamente.")


# Uso polim√≥rfico

class Figura(ABC):
    def __init__(self, nombre: str, color: str = "Negro") -> None:
        self.nombre = nombre
        self.color = color

    @abstractmethod
    def calcular_area(self) -> float:
        raise NotImplementedError

    @abstractmethod
    def calcular_perimetro(self) -> float:
        raise NotImplementedError

    def mostrar_info(self) -> None:
        print(f"Figura: {self.nombre}")
        print(f"Color: {self.color}")
        print(f"Area: {self.calcular_area():.2f}")
        print(f"Perimetro: {self.calcular_perimetro():.2f}")
