Conceptos Teóricos

¿Qué es Python?

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general. Fue creado por Guido van Rossum y lanzado por primera vez en 1991. Sus características principales incluyen:

  • Sintaxis limpia y legible: El código Python se lee como pseudocódigo en inglés
  • Tipado dinámico: No necesitas declarar el tipo de variable
  • Multiparadigma: Soporta POO, programación funcional y procedural
  • Baterías incluidas: Biblioteca estándar extensa
  • Código abierto: Gratis y con una comunidad activa
💡 Dato Curioso: El nombre "Python" viene de los Monty Python, no de la serpiente. Guido van Rossum era fan del show de comedia británico.

El Ecosistema de Python

Python tiene uno de los ecosistemas más grandes de cualquier lenguaje de programación:

  • PyPI (Python Package Index): Más de 500,000 paquetes disponibles
  • pip: Gestor de paquetes estándar para instalar bibliotecas
  • Virtual environments: venv, conda, poetry para aislar proyectos
  • Herramientas de desarrollo: VSCode, PyCharm, Jupyter Notebooks

Variables y Asignación

Una variable es un nombre que referencia un valor en memoria. En Python, las variables se crean mediante asignación:

# Asignación básica
nombre: str = "Juan"
edad: int = 25
altura: float = 1.75
es_estudiante: bool = True

Python usa tipado dinámico, lo que significa que el tipo de una variable se determina en tiempo de ejecución:

x: int = 10        # x es int
x = "Hola"          # ahora x es str (Python infiere el tipo)
x = 3.14            # ahora x es float
📝 Nota: Aunque Python permite cambiar el tipo de una variable, esto se considera una mala práctica. Mantén tipos consistentes.

Input/Output Básico

Para interactuar con el usuario, Python proporciona dos funciones fundamentales:

# Output: imprimir en pantalla
print("Hola, mundo!")
print("El valor de x es:", x)

# Input: leer desde el teclado (siempre retorna str)
nombre = input("¿Cuál es tu nombre? ")
print("Mucho gusto,", nombre)

# Convertir input a número
edad = int(input("¿Cuántos años tienes? "))
altura = float(input("¿Cuánto mides? "))

Ejemplo 5: Conversión de Temperatura

# temperatura.py
# Convertir Celsius a Fahrenheit
celsius = float(input("Ingresa temperatura en Celsius: "))

fahrenheit = (celsius * 9/5) + 32

print(f"\n{celsius}°C = {fahrenheit}°F")

# Convertir Fahrenheit a Celsius
fahrenheit = float(input("Ingresa temperatura en Fahrenheit: "))

celsius = (fahrenheit - 32) * 5/9

print(f"{fahrenheit}°F = {celsius:.2f}°C")

Ejemplo 6: Cálculo de Promedio

# promedio.py
print("Cálculo de promedio de 3 notas")

nota1 = float(input("Ingresa nota 1: "))
nota2 = float(input("Ingresa nota 2: "))
nota3 = float(input("Ingresa nota 3: "))

promedio = (nota1 + nota2 + nota3) / 3

print(f"\n--- Resultado ---")
print(f"Nota 1: {nota1}")
print(f"Nota 2: {nota2}")
print(f"Nota 3: {nota3}")
print(f"Promedio: {promedio:.2f}")

if promedio >= 3.0:
    print("¡Aprobado!")
else:
    print("Reprobado")

Buenas Prácticas

Convenciones de Nomenclatura (PEP 8)

  • Variables y funciones: snake_case (minúsculas con guiones bajos) 
    mi_variable = 10
nombre_usuario = "Juan"
def calcular_promedio():
  • Constantes: UPPER_CASE (mayúsculas con guiones bajos) 
    MAX_INTENTOS = 3
PI = 3.14159
  • Nombres descriptivos: Usa nombres que expliquen el propósito 
    # ❌ Mal
x = 25
d = "Juan"

# ✅ Bien
edad = 25
nombre = "Juan"

F-Strings para Formato

Los f-strings (Python 3.6+) son la forma recomendada de formatear strings:

# ✅ Bien (Python 3.6+)
nombre = "Ana"
edad = 25
print(f"Hola, {nombre}. Tienes {edad} años")

# ⚠️ Funciona pero es más verboso
print("Hola, {}. Tienes {} años".format(nombre, edad))

# ❌ Antiguo, evitar
print("Hola, %s. Tienes %d años" % (nombre, edad))

Tipado Explícito (Type Hints)

Aunque es opcional, es buena práctica indicar los tipos:

# Con type hints (recomendado)
nombre: str = "Juan"
edad: int = 25
altura: float = 1.75
es_estudiante: bool = True

Errores Comunes a Evitar

❌ No Hacer

  • Usar nombres de una sola letra (excepto en loops)
  • Cambiar el tipo de una variable durante la ejecución
  • Usar palabras reservadas como nombres de variable (print, list, type)
  • Olvidar convertir el input (input() siempre retorna str)

✅ Sí Hacer

  • Usar nombres descriptivos y en español (o inglés consistente)
  • Convertir explícitamente el input: int(input(...))
  • Usar f-strings para formatear salida
  • Agregar comentarios para explicar código complejo

Ejercicio Guidado

🎯 Ejercicio: Tu Primer Script Python

Vamos a crear un script que calcule el índice de masa corporal (IMC) de una persona.

Paso 1: Crear el archivo

  1. Abre VSCode
  2. Crea un nuevo archivo: imc.py
  3. Guarda el archivo en tu carpeta de trabajo

Paso 2: Escribir el código

# imc.py
"""
Calculadora de Índice de Masa Corporal (IMC)

Fórmula: IMC = peso / altura²
Donde:
- peso está en kilogramos
- altura está en metros
"""

print("=== Calculadora de IMC ===")

# Solicitar datos al usuario
nombre = input("¿Cuál es tu nombre? ")
peso = float(input("Ingresa tu peso en kg: "))
altura = float(input("Ingresa tu altura en metros: "))

# Calcular IMC
imc = peso / (altura ** 2)

# Mostrar resultado
print(f"\n--- Resultado para {nombre} ---")
print(f"Peso: {peso} kg")
print(f"Altura: {altura} m")
print(f"IMC: {imc:.2f}")

# Clasificación según la OMS
if imc < 18.5:
    clasificacion = "Bajo peso"
elif imc < 25:
    clasificacion = "Peso normal"
elif imc < 30:
    clasificacion = "Sobrepeso"
else:
    clasificacion = "Obesidad"

print(f"Clasificación: {clasificacion}")

Paso 3: Ejecutar el script

# Desde VSCode: Ctrl+F5 (Run Python File)

# O desde terminal:
python imc.py

# O en macOS/Linux:
python3 imc.py

Paso 4: Probar con diferentes valores

Nombre Peso (kg) Altura (m) IMC esperado
Juan 70 1.75 22.86
Maria 55 1.60 21.48
Carlos 90 1.70 31.14
⚠️ Reto adicional: Modifica el script para que también calcule el peso ideal según la fórmula: peso_ideal = IMC_deseado * altura² (usando IMC = 22 como peso saludable).
