Conceptos Te√≥ricos

1. Listas (Lists)

Las listas son colecciones ordenadas y mutables de elementos. Pueden contener diferentes tipos de datos.

Creaci√≥n de Listas

# Lista vac√≠a
lista_vacia = []

# Lista con elementos
numeros = [1, 2, 3, 4, 5]
nombres = ["Ana", "Juan", "Mar√≠a"]
mixta = [1, "hola", 3.14, True]

# Lista con range()
pares = list(range(0, 10, 2))  # [0, 2, 4, 6, 8]

Operaciones con Listas

numeros = [10, 20, 30, 40, 50]

# Acceder por √≠ndice (empezando en 0)
print(numeros[0])   # 10
print(numeros[-1])  # 50 (√∫ltimo elemento)

# Slicing [inicio:fin:paso]
print(numeros[1:4])   # [20, 30, 40]
print(numeros[:3])    # [10, 20, 30]
print(numeros[::2])   # [10, 30, 50] (pares)

# Longitud
print(len(numeros))  # 5

# Modificar elementos
numeros[0] = 100     # [100, 20, 30, 40, 50]

# M√©todos comunes
numeros.append(60)        # Agrega al final: [..., 60]
numeros.insert(1, 15)     # Inserta en posici√≥n 1
numeros.extend([70, 80])  # Extiende con otra lista
ultimo = numeros.pop()    # Elimina y retorna el √∫ltimo
numeros.remove(30)        # Elimina el primer 30 encontrado
numeros.sort()            # Ordena la lista
numeros.reverse()         # Invierte la lista
indice = numeros.index(20)  # Retorna √≠ndice de 20
cuenta = numeros.count(20)  # Cuenta cu√°ntos 20 hay

2. Tuplas (Tuples)

Las tuplas son colecciones ordenadas e inmutables. Una vez creadas, no pueden modificarse.

# Crear tupla
coordenadas = (10, 20)
rgb = (255, 128, 0)
sin_parentesis = 1, 2, 3  # Tambi√©n es una tupla

# Tupla vac√≠a (nota la coma)
vacia = ()

# Tupla de un solo elemento (nota la coma)
singleton = (42,)

# Acceder como listas
print(coordenadas[0])  # 10
print(rgb[1:3])        # (128, 0)

# Desempaquetado de tupla
x, y = coordenadas
r, g, b = rgb

# Desempaquetado con *
primero, *resto = [1, 2, 3, 4, 5]
# primero = 1, resto = [2, 3, 4, 5]
üìù ¬øCu√°ndo usar tuplas vs listas?
  • Tupla: Datos que no deben cambiar (coordenadas, configuraciones, valores constantes)
  • Lista: Datos que pueden cambiar (colecciones din√°micas, secuencias modificables)

3. Diccionarios (Dictionaries)

Los diccionarios almacenan pares clave-valor. Son mutables y no mantienen orden (antes de Python 3.7).

# Crear diccionario
vacio = {}
usuario = {
    "nombre": "Juan",
    "edad": 25,
    "email": "juan@example.com"
}

# Acceder por clave
print(usuario["nombre"])        # "Juan"
print(usuario.get("edad"))      # 25
print(usuario.get("pais", "Colombia"))  # "Colombia" (valor por defecto)

# Modificar y agregar
usuario["edad"] = 26            # Modificar
usuario["pais"] = "Colombia"    # Agregar nueva clave

# M√©todos √∫tiles
claves = usuario.keys()         # dict_keys(['nombre', 'edad', 'email', 'pais'])
valores = usuario.values()      # dict_values(['Juan', 26, ...])
items = usuario.items()         # dict_items([('nombre', 'Juan'), ...])

# Eliminar
del usuario["email"]            # Eliminar clave
valor = usuario.pop("edad")     # Eliminar y retornar valor

# Verificar si existe clave
if "nombre" in usuario:
    print("Clave 'nombre' existe")

4. Sets (Conjuntos)

Los sets son colecciones desordenadas de elementos √∫nicos. No permiten duplicados.

# Crear set
vacio = set()
numeros = {1, 2, 3, 4, 5}
con_duplicados = {1, 2, 2, 3, 3, 3}  # {1, 2, 3}

# Desde lista
lista = [1, 2, 2, 3, 3, 3]
unicos = set(lista)  # {1, 2, 3}

# M√©todos
numeros.add(6)           # Agregar elemento
numeros.remove(3)        # Eliminar (error si no existe)
numeros.discard(10)      # Eliminar (sin error si no existe)
numeros.pop()            # Eliminar elemento arbitrario

# Operaciones matem√°ticas
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

union = a | b            # {1, 2, 3, 4, 5, 6}
interseccion = a & b     # {3, 4}
diferencia = a - b       # {1, 2}
diferencia_sim = a ^ b   # {1, 2, 5, 6}

# Verificar relaci√≥n
a.issubset(b)            # a est√° contenido en b
a.issuperset(b)          # a contiene a b
a.isdisjoint(b)          # a y b no comparten elementos

5. Comprensiones de Lista (List Comprehensions)

Las comprensiones crean listas de manera concisa y "pythonica".

# B√°sico
numeros = [1, 2, 3, 4, 5]
cuadrados = [x**2 for x in numeros]  # [1, 4, 9, 16, 25]

# Con condici√≥n
pares = [x for x in numeros if x % 2 == 0]  # [2, 4]

# Con if-else
clases = ["par" if x % 2 == 0 else "impar" for x in numeros]

# Anidada
matriz = [[i*j for j in range(3)] for i in range(3)]
# [[0, 0, 0], [0, 1, 2], [0, 2, 4]]
üí° Consejo: Las comprensiones son m√°s r√°pidas y legibles que usar map() o bucles for tradicionales para crear listas.

Resumen de Estructuras de Datos

Estructura Ordenada Mutable Permite Duplicados Uso T√≠pico
list S√≠ S√≠ S√≠ Colecciones secuenciales
tuple S√≠ No S√≠ Datos constantes, retorno de funciones
dict S√≠* (3.7+) S√≠ Claves √∫nicas Mapeos clave-valor
set No S√≠ No Eliminaci√≥n de duplicados, operaciones matem√°ticas

# Sistema simple de gesti√≥n de tareas

tareas = [
    {"id": 1, "titulo": "Configurar Python", "completada": True},
    {"id": 2, "titulo": "Aprender listas", "completada": True},
    {"id": 3, "titulo": "Practicar diccionarios", "completada": False},
    {"id": 4, "titulo": "Crear proyecto TaskFlow", "completada": False}
]

# Filtrar tareas pendientes
pendientes = [t for t in tareas if not t["completada"]]
print(f"Tareas pendientes: {len(pendientes)}")

# Filtrar tareas completadas
completadas = [t for t in tareas if t["completada"]]
print(f"Tareas completadas: {len(completadas)}")

# Buscar tarea por ID
def buscar_tarea_por_id(tareas: list, tarea_id: int) -> dict | None:
    """Busca una tarea por su ID."""
    for tarea in tareas:
        if tarea["id"] == tarea_id:
            return tarea
    return None

# Marcar tarea como completada
def completar_tarea(tareas: list, tarea_id: int) -> bool:
    """Marca una tarea como completada."""
    tarea = buscar_tarea_por_id(tareas, tarea_id)
    if tarea:
        tarea["completada"] = True
        return True
    return False

# Agregar nueva tarea
def agregar_tarea(tareas: list, titulo: str) -> dict:
    """Agrega una nueva tarea."""
    nuevo_id = max([t["id"] for t in tareas], default=0) + 1
    nueva_tarea = {
        "id": nuevo_id,
        "titulo": titulo,
        "completada": False
    }
    tareas.append(nueva_tarea)
    return nueva_tarea

# Uso
print("\n--- Tareas Pendientes ---")
for tarea in pendientes:
    print(f"  [{tarea['id']}] {tarea['titulo']}")

# Completar una tarea

import math

def distancia_desde_origen(punto: tuple) -> float:
    """Calcula la distancia de un punto desde el origen."""
    x, y = punto
    return math.sqrt(x**2 + y**2)

# gestor_contactos.py
"""
Gestor de Contactos
Sistema para gestionar una lista de contactos con CRUD completo.
"""

from typing import TypedDict

class Contacto(TypedDict):
    """Estructura de un contacto."""
    id: int
    nombre: str
    telefono: str
    email: str
    tags: list[str]

# Lista de contactos
contactos: list[Contacto] = []

def generar_id() -> int:
    """Genera un nuevo ID √∫nico."""
    if len(contactos) == 0:
        return 1
    return max(c["id"] for c in contactos) + 1

def agregar_contacto(
    nombre: str,
    telefono: str,
    email: str,
    tags: list[str] | None = None
) -> Contacto:
    """Agrega un nuevo contacto."""
    nuevo_contacto: Contacto = {
        "id": generar_id(),
        "nombre": nombre,
        "telefono": telefono,
        "email": email,
        "tags": tags or []
    }
    contactos.append(nuevo_contacto)
    return nuevo_contacto

def listar_contactos() -> None:
    """Lista todos los contactos."""
    if len(contactos) == 0:
        print("\nüì≠ No hay contactos registrados.")
        return

    print(f"\nüìá Contactos ({len(contactos)}):")
    for contacto in contactos:
        tags_str = ", ".join(contacto["tags"]) if contacto["tags"] else "Sin tags"
        print(f"  [{contacto['id']}] {contacto['nombre']}")
        print(f"      Tel: {contacto['telefono']}")
        print(f"      Email: {contacto['email']}")
        print(f"      Tags: {tags_str}")

def buscar_contacto(criterio: str) -> list[Contacto]:
    """Busca contactos por nombre, email o tel√©fono."""
    resultados = []
    criterio_lower = criterio.lower()

    for contacto in contactos:
        if (criterio_lower in contacto["nombre"].lower() or
            criterio_lower in contacto["email"].lower() or
            criterio_lower in contacto["telefono"]):
            resultados.append(contacto)

    return resultados

def buscar_por_tag(tag: str) -> list[Contacto]:
    """Busca contactos por tag."""
    tag_lower = tag.lower()
    return [
        c for c in contactos
        if any(tag_lower in t.lower() for t in c["tags"])
    ]

def editar_contacto(
    contacto_id: int,
    nombre: str | None = None,
    telefono: str | None = None,
    email: str | None = None,
    tags: list[str] | None = None
) -> bool:
    """Edita un contacto existente."""
    for contacto in contactos:
        if contacto["id"] == contacto_id:
            if nombre:
                contacto["nombre"] = nombre
            if telefono:
                contacto["telefono"] = telefono
            if email:
                contacto["email"] = email
            if tags is not None:
                contacto["tags"] = tags
            return True
    return False

def eliminar_contacto(contacto_id: int) -> bool:
    """Elimina un contacto por ID."""
    for i, contacto in enumerate(contactos):
        if contacto["id"] == contacto_id:
            contactos.pop(i)
            return True
    return False

def obtener_tags_unicos() -> list[str]:
    """Retorna todos los tags √∫nicos de los contactos."""
    todos_los_tags = []
    for contacto in contactos:
        todos_los_tags.extend(contacto["tags"])
    return sorted(set(todos_los_tags))

def mostrar_menu() -> None:
    """Muestra el men√∫ principal."""
    print("\n" + "="*50)
    print("üìá GESTOR DE CONTACTOS")
    print("="*50)
    print("1. Listar contactos")
    print("2. Agregar contacto")
    print("3. Buscar contacto")
    print("4. Buscar por tag")
    print("5. Editar contacto")
    print("6. Eliminar contacto")
    print("7. Ver todos los tags")
    print("8. Salir")

def main() -> None:
    """Funci√≥n principal."""
    # Agregar algunos contactos de ejemplo
    agregar_contacto(
        "Juan P√©rez",
        "3001234567",
        "juan@example.com",
        ["trabajo", "amigos"]
    )
    agregar_contacto(
        "Mar√≠a Garc√≠a",
        "3009876543",
        "maria@example.com",
        ["familia"]
    )

    while True:
        mostrar_menu()
        opcion = input("\nSelecciona una opci√≥n (1-8): ")

        if opcion == "1":
            listar_contactos()

        elif opcion == "2":
            print("\n--- Nuevo Contacto ---")
            nombre = input("Nombre: ")
            telefono = input("Tel√©fono: ")
            email = input("Email: ")
            tags_input = input("Tags (separados por coma): ")
            tags = [t.strip() for t in tags_input.split(",") if t.strip()]

            contacto = agregar_contacto(nombre, telefono, email, tags)
            print(f"\n‚úÖ Contacto agregado con ID: {contacto['id']}")

        elif opcion == "3":
            criterio = input("\nIngresa nombre, email o tel√©fono a buscar: ")
            resultados = buscar_contacto(criterio)

            if resultados:
                print(f"\nüîç Resultados ({len(resultados)}):")
                for contacto in resultados:
                    print(f"  [{contacto['id']}] {contacto['nombre']}")
            else:
                print("\n‚ùå No se encontraron contactos.")

        elif opcion == "4":
            tags = obtener_tags_unicos()
            if not tags:
                print("\n‚ùå No hay tags registrados.")
                continue

            print(f"\nTags disponibles: {', '.join(tags)}")
            tag = input("Ingresa el tag a buscar: ")
            resultados = buscar_por_tag(tag)

            if resultados:
                print(f"\nüè∑Ô∏è Contactos con tag '{tag}':")
                for contacto in resultados:
                    print(f"  [{contacto['id']}] {contacto['nombre']}")
            else:
                print(f"\n‚ùå No hay contactos con el tag '{tag}'.")

        elif opcion == "5":
            listar_contactos()
            contacto_id = input("\nIngresa el ID del contacto a editar: ")

            try:
                contacto_id = int(contacto_id)
            except ValueError:
                print("‚ùå ID inv√°lido.")
                continue

            nombre = input(f"Nuevo nombre (Enter para mantener): ") or None
            telefono = input(f"Nuevo tel√©fono (Enter para mantener): ") or None
            email = input(f"Nuevo email (Enter para mantener): ") or None
            tags_input = input(f"Nuevos tags (Enter para mantener): ")

            tags = None
            if tags_input:
                tags = [t.strip() for t in tags_input.split(",") if t.strip()]

            if editar_contacto(contacto_id, nombre, telefono, email, tags):
                print("\n‚úÖ Contacto actualizado")
            else:
                print("\n‚ùå Contacto no encontrado")

        elif opcion == "6":
            listar_contactos()
            contacto_id = input("\nIngresa el ID del contacto a eliminar: ")

            try:
                contacto_id = int(contacto_id)
            except ValueError:
                print("‚ùå ID inv√°lido.")
                continue

            if eliminar_contacto(contacto_id):
                print("\n‚úÖ Contacto eliminado")
            else:
                print("\n‚ùå Contacto no encontrado")

        elif opcion == "7":
            tags = obtener_tags_unicos()
            if tags:
                print(f"\nüè∑Ô∏è Tags: {', '.join(tags)}")
            else:
                print("\n‚ùå No hay tags registrados.")

        elif opcion == "8":
            print("\n¬°Gracias por usar el Gestor de Contactos!")
            break

        else:
            print("\n‚ùå Opci√≥n no v√°lida. Intenta de nuevo.")

if __name__ == "__main__":
    main()
